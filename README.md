# CampusCred

A Centralized Internship & Placement Tracking System.

## Getting Started

### Prerequisites
- Node.js (v18+)
- Java 17+
- Maven (optional, wrapper included)

### Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

### Backend Setup
```bash
cd backend
./mvnw spring-boot:run
```

## Tech Stack
- **Frontend**: React, Vite, Tailwind CSS, shadcn/ui, Firebase
- **Backend**: Spring Boot, Java, JPA, H2 Database

## Project Structure
- `/frontend`: React application.
- `/backend`: Spring Boot application.
- `firebase.js`: Firebase configuration (requires API keys).
